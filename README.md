## üèó About the Project

This project was developed as a freelance commission to modernize the digital presence of a Saudi Arabian cleaning company. The goal was to create a trust-inspiring, high-converting landing page that caters to both local (Arabic speaking) and expat (English speaking) customers.

The platform showcases various services such as:
* **Residential & Commercial Cleaning**
* **Deep Cleaning & Sanitization**
* **Hourly Cleaning Services and much more**

The site emphasizes performance, accessibility, and smooth user interactions using the latest Next.js ecosystem.
